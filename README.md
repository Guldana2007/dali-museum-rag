# Dalí Museum — RAG-based AI System

A small RAG prototype that answers questions about the Dalí Museum using:
- Chroma vector database
- Embedding API
- LLM client
- Jupyter Notebook pipeline
- Optional Streamlit UI

## Repository Structure

notebook/dali_rag.ipynb — main RAG pipeline

data/dali_chunks.json — dataset used for building the vector DB

requirements.txt — Python dependencies

app_streamlit.py — optional UI

## How to Run
1. Install dependencies:  
   `pip install -r requirements.txt`
2. Open `notebook/dali_rag.ipynb` in Jupyter.
3. Run all cells from top to bottom.
